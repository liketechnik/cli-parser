class Parser (source)

This class parses the passed arguments (from the cli) for the specified parameters and returns them on request.

The values are internally stored and can be retrieved via their id with the correct methods for the value's data type (e. g. getStringArgumentValue for Strings).

Parameters

arguments - The arguments to parse (usually from cli).

argumentPrefix - The prefix for long (default) argument forms (for example '--file=test.txt').

argumentSecondaryPrefix - The prefix for short argument forms (for example '-f test.txt').

argumentValueSeparator - The separator between argument and value for the long form.

parameters - The parameters to parse from the arguments.
