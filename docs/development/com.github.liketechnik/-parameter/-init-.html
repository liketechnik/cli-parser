Parameter()

This class represents a command line parameter.

name and shortName are used to identify the passed arguments and match their values to the correct parameters. defaultValue must have the same type as type is stating; it is used when the parameter is not specified on the cli. type defines which (java/kotlin) data type the value of the parameter should have. id identifies the parameter amongst others in a parser and must be unique
